Habilidades

HTML5

Sigla para HyperText Markup Language — Linguagem de Marcação de Hipertexto —, o HTML é o componente base da web. Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.

CSS3

CSS é a sigla para Cascading Style Sheets, ou seja, Folhas de Estilo em Cascatas. É uma maneira de dar estilo ao código criado por linguagens como HTML, XML ou XHTML

JavaScript

JavaScript é uma linguagem de programação de alto nível voltada para aplicações web, podendo manipular componentes de páginas web.

React

O React é a biblioteca mais popular do JavaScript e é usada para construir uma interface de usuário (IU). Ela oferece uma resposta excelente para o usuário adicionar comandos usando um novo método de renderizar sites.

Git

O Git é um sistema de controle de versão distribuído gratuitamente e de código aberto projetado para lidar com tudo, desde projetos pequenos a muito grandes com velocidade e eficiência.

GitHub

GitHub é uma plataforma de hospedagem de código para controle de versão e colaboração. Ele permite que muitas pessoas trabalhem juntas em projetos de qualquer lugar do mundo.
